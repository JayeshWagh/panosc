Jupyter

As a teacher, I would like for the students to be able to answer an exercise in Jupyter

 

In the following the student launches a Jupyter notebook from or within EP

1.     Do something in the notebook and write the result in a box in the LMS to check for correctness

2.     Do something in the notebook and (semi-)automatically communicate result to LMS to check for correctness

3.     Do something in the notebook, check immediately in the notebook that the result is correct and communicate result or grade back to LMS

4.     Do something in the notebook which then is stored so that the teacher can grade it. The LMS system is automatically updated.
